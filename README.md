# SmartTrac

SmartTrac is a backend product inventory management system built using FastAPI, PostgreSQL, and SQLAlchemy. It provides RESTful APIs for managing products and demonstrates core backend development concepts.

## Features

* View all products
* Add new products
* Update existing products
* Delete products
* PostgreSQL database integration
* SQLAlchemy ORM implementation
* FastAPI dependency injection
* Frontend integration through CORS

## Tech Stack

* Python
* FastAPI
* PostgreSQL
* SQLAlchemy
* Pydantic

## What I Learned

* Building REST APIs with FastAPI
* Database integration using PostgreSQL
* ORM concepts with SQLAlchemy
* CRUD operations and API design
* Request handling and data validation
* Backend application architecture
